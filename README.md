
#### Examples
```python
>>> import temp
>>> temp.tempdir() # dir
'/var/folders/rv/gy6_gnfs1d7_pd1518p27tsr0000gn/T/tmpqlLDxb'

>>> temp.tempfile() # file
'/var/folders/rv/gy6_gnfs1d7_pd1518p27tsr0000gn/T/tmpsctFHJ'

>>> temp.TMPDIR
'/var/folders/rv/gy6_gnfs1d7_pd1518p27tsr0000gn/T'
```
